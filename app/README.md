# Gói kế hoạch Video Workspace V36

Gói này chỉ chứa tài liệu thiết kế và prompt triển khai. Chưa sửa chức năng trong source.

Thứ tự dùng:

1. Chép `VIDEO_WORKSPACE_V36_MASTER_PLAN.md` vào project root.
2. Mở Codex tại project đầy đủ.
3. Dán nội dung `CODEX_PASS_00_01_PROMPT.md`.
4. Sau khi Codex hoàn thành PASS 00 + 01, đối chiếu `V36_ACCEPTANCE_CHECKLIST.md` trước khi làm PASS 02.
